# Dedalus Marketplace Info

## Short Description
OpenStreetMap geocoding, POI search, and routing

## Long Description
Forward and reverse geocode addresses using Nominatim, search for points of interest via Overpass API, and calculate routes for driving, cycling, or walking via OSRM. Respects OSM usage policies. Tools: geocode, reverse_geocode, poi_search, route.

## Use Cases
- Maps
- Travel
